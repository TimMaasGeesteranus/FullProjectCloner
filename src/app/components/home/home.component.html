What is FullProjectCloner?

FullProjectCloner makes it possible to clone a repository to selected recipients. Cloning will include the full GitHub repository including its issues and project board.

How does it work?

  • The user authenticates with GitHub.
  • The user installs the GitHub app on their GitHub account.
  • The user selects a specific GitHub project.
  • The user chooses recipient(s).
  • A request e-mail will be sent out to the recipient(s).
  • The recipient accepts the link sent to their email address.
  • The recipient authenticates with GitHub.
  • The recipient installs the GitHub app on their GitHub account.
  • The recipient chooses to clone the project to their GitHub account.
